shGoogleGeocoding

  1. service in module SuhExternal

Handles google geocoding API requests

Methods

  • init();

    initializes Google maps API

  • postInit();

    performs post initialization of the Google Maps API.

  • ready();

    returns the initialization promise of the API initialization.

  • geocodeAddress();

    geocode a given address using the name of the address

  • reverseGeocode(latlng);

    [description]

    latlng object

Properties

  • _initialized

    indicates whether the API is appropriately loaded or not

  • coder

    holds a reference to the service geocoder used to contact Google Maps servers

  • coder

    holds a reference to the map object used to query Google Maps servers

  • apiKey

    holds the API key used to communicate with Google Maps servers
